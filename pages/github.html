GitHub 加速下载

使用方法：
  1. 打开你要下载的 GitHub 仓库页面
  2. 点击右侧的绿色按钮 "Code" > "Download ZIP"
  3. 等浏览器弹出下载框后复制下载框中的链接地址并粘贴到以上输入框
  4. 点击加速下载
命令行clone：
  1. git clone https://tegic.gitclone.workers.dev/https://github.com/teg1c/project.git
